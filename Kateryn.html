Mi explicacion del codigo C++

Comeznariamos con poner un include, als librerias y posteriormente abririamos el main()

despues dentro del main comenzariamos por colocar un printf y dentro del printf se colocaria "dame una oracion o enunciado" o algo parecido

posteriormente de eso colocaremos un scanf que guardara la cadena de caracteres en una variable que llamaremos como mas nos guste por ejemplo "cuenta"

una ves tengamos la variable guardada usaremos un if para comparar que sea un string

si es asi le dejaremos seguir y con una funcion que contara el numero de palabras que tiene la cadena de caracteres

y por ultimo le imprimiremos el numero de la cadena de caracteres si lo introducido es un string

si es otra cosa usaremos un else el cual imprimira que esta mal y le dejara volver a intentarlo 2 veces mas

esto se repetira 2 veces mas una ves que el usuario se gaste las 3 oportunidades, se cerrara el progama sin dejarte volver a intentarlo
